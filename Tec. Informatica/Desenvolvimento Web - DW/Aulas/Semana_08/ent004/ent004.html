Resident Evil 4 - Guia Rápido 🎮

Bem-vindo ao guia rápido para dominar os segredos de "Resident Evil 4"! Prepare-se para aventura © 2025.

Comandos Essenciais de Movimentação

  • Mover para Cima: Tecla W / Console: ↑
  • Mover para Baixo: Tecla S / Console: ↓
  • Mover para Esquerda: Tecla A / Console: ←
  • Mover para Direita: Tecla D / Console: →
  • Pular / Confimar: Tecla Barra de Espaço ou Enter / Console: X (X)
  • Ação Secundária / Cancelar: Tecla CTRL / Console: ° (Bola/Círculo)
  • Menu / Opções: Tecla ESC / Console: ▲ (Triângulo)
  • Inventário: Tecla I / Console: ■ (Quadrado)

Sequência de Ataque Especial (Combo "Fúria Pixelada")

  1. Pressione ■, ■ (duas vezes rapidamente)
  2. Em seguida, pressione ° com precisão
  3. Finalize segundo R2 e apertando X

Glossário de Itens do Jogo

Cristal de Energia 💎

Restaura 100 pontos de energia da sua nave. Use com sabedoria!

Escudo Defletor Mk.II 🛡

Oferece proteção temporária contra todos os tipos de dano. Dura 30 segundos.

XP Doubler ✨

Dobra a quantidade de Pontos de Experiência ganhos por 5 minutos.
